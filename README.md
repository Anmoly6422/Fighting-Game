# 🔥 FightBox Pro

**Fire. Reflex. Domination.**

FightBox Pro is a fast-paced, browser-based 1v1 fighting game where you battle an AI opponent using quick reflexes, timed attacks, and strategic blocking.

🎮 **Live Demo:** [clownfighter.netlify.app](https://clownfighter.netlify.app/)

---

## 🕹️ Gameplay

Face off against a bot in a best-of-skill fight. Manage your health, land fireballs, time your blocks, and outplay your opponent before they outplay you.

### Controls
| Key | Action |
|-----|--------|
| `A` / `D` | Move Left / Right |
| `W` | Jump |
| `S` | Fireball |
| `SHIFT` | Block |

### Difficulty Levels
- 🟢 Easy
- 🟡 Medium
- 🔴 Hard

### Features
- ⚔️ Real-time 1v1 combat against an AI bot
- 🔥 Fireball ranged attacks
- 🛡️ Blocking mechanic to reduce/avoid damage
- ❤️ Health bar system (You vs Bot)
- 🔁 Replay option after each match
- 📊 Adjustable difficulty for varied challenge

## 🚀 Getting Started (for developers)

Clone the repo and open `index.html` in your browser — no build steps required.

\`\`\`bash

cd fightbox-pro
open index.html
